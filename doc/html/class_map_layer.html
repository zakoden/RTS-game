◆ Generate()

virtual void MapLayer::Generate ( uint64_t  seed)
pure virtual

Generates map based on a seed. This function must not be called in constructor. It will be called in GameMap. This function must be deterministic, i.e. if Generate is called with the same seed, then the results should be the same. As the result, Generate must fill blocks_ array with blocks that will be used to generate map.

Замещается в UndergroundLayer и SurfaceLayer.
